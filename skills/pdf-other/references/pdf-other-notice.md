# other notice

> **前置条件：** 先阅读 [`../pdf-shared/SKILL.md`](../../pdf-shared/SKILL.md)。

查看系统公告和首页配置。

## 命令

```bash
pdf-cli other notice

# JSON 格式
pdf-cli other notice --format json
```

## 参数

| 参数 | 必填 | 说明 |
|------|------|------|
| `--format <type>` | 否 | 输出格式 |

## API

- GET `user/config/homepage`
- 不需要登录
